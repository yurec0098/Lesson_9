## Commands

* copy_file		*copy_file "orig_file_path" "new_file_path"* 
* copy_dir		*copy_dir "orig_dir_path" "new_dir_path"* 
* del_file		*del_file "file_path"* 
* del_dir		*del_dir "dir_path"* 
* cd			*cd "path"*
